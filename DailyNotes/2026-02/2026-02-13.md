- [x] **算法**
  - **手写快排 (Quick Sort)**
    - **核心**：Partition (分区)。
      - 选 pivot (通常选第一个或随机)。
      - 双指针 swap，左边 < pivot，右边 > pivot。
      - 递归 left, right。
  - **课程表 (Topology Sort)**
    - **思路**：入度表 (Indegree) + 邻接表 (Adjacency List) + 队列 (Queue)。
    - 1. 统计所有课程先修课数量 (入度)。
    - 2. 入度为 0 入队。
    - 3. 出队 count++，将后续课程入度 -1，若为 0 入队。
    - 4. 若 count == numCourses，无环。
  - **接雨水 (Hard)**
    - **方法**：双指针 (Two Pointers)。
      - 维护 maxLeft, maxRight。
      - 左右指针向中间收缩。
      - `water += min(maxLeft, maxRight) - height[i]`。
  - **耗时**：2h
