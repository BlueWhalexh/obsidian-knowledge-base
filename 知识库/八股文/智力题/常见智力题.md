# 常见智力题

## 一、猴子搬香蕉

### 问题

一只猴子面前有 100 根香蕉，猴子距离家 50 米。猴子一次最多搬 50 根香蕉，但每走 1 米就要吃 1 根香蕉。问猴子最多能搬多少根香蕉回家？

### 思路

关键在于：猴子可以分段搬运，先把香蕉运到中间某个点，再从中间点运回家。

**分段策略**：当香蕉数量 > 50 根时，需要多次往返搬运。

**第一段（起点到 A 点）**：

- 100 根香蕉，每次搬 50 根
- 需要 3 次才能搬完（去→回→去 = 3 趟）
- 每趟每米消耗 1 根，3 趟每米消耗 3 根
- 搬到 A 点时：100 - 3 × A = 50（搬完一次只剩 50 根时停止）
- A = (100 - 50) / 3 = 16.67，取整 A = 16 米
- 搬到 A 点剩余：100 - 3 × 16 = 52 根（但最多搬 50 根，需要调整）

**更精确的分析**：

第一段（0→16 米）：100 根，3 趟/米，消耗 48 根，到达 16 米处剩余 52 根
- 但一次只能搬 50 根，所以实际需要调整

**重新分析**：

```
第一段：0 → X 米，每次 50 根
  100 根需要 3 趟（去、回、去）：每米消耗 3 根
  当剩余 ≤ 50 根时，只需 1 趟
  X = (100 - 50) / 3 ≈ 16.67，取 X = 16
  到达 16 米处：100 - 3 × 16 = 52 根
  但一次搬 50 根，需要 2 趟（去、回、去 = 3 趟一次搬 50 和 2）
  → 调整：第一段走 16 米，消耗 48 根，剩余 52 根

第二段：16 → Y 米，52 根需要 2 趟（每米消耗 2 根）
  当剩余 ≤ 50 根时，只需 1 趟
  Y = 16 + (52 - 50) / 2 = 17
  到达 17 米处：52 - 2 × 1 = 50 根

第三段：17 → 50 米，50 根只需 1 趟
  消耗：50 - 17 = 33 根
  到达 50 米处：50 - 33 = 17 根
```

### 答案

**猴子最多能搬 16 根香蕉回家。**

## 二、白鼠找毒药

### 问题

有 1000 瓶药水，其中 1 瓶是毒药。毒药会在 24 小时后致死。你有 10 只白鼠，如何在 24 小时内找出哪瓶是毒药？

### 思路

**二进制编码法**。

1000 瓶药水需要 10 位二进制来编号（2^10 = 1024 > 1000）。

```
第 1 瓶：0000000001
第 2 瓶：0000000010
第 3 瓶：0000000011
...
第 1000 瓶：1111101000
```

**分配规则**：

- 第 1 只白鼠喝第 1 位为 1 的所有药水（第 1, 3, 5, 7, ... 瓶）
- 第 2 只白鼠喝第 2 位为 1 的所有药水（第 2, 3, 6, 7, ... 瓶）
- ...
- 第 10 只白鼠喝第 10 位为 1 的所有药水

**24 小时后**：

- 死掉的白鼠对应位为 1
- 存活的白鼠对应位为 0
- 组合起来就是毒药的编号

**举例**：

```
假设第 7 瓶是毒药（0000000111）
第 1 只白鼠死了（第 1 位 = 1）
第 2 只白鼠死了（第 2 位 = 1）
第 3 只白鼠死了（第 3 位 = 1）
其他白鼠存活
→ 编码 = 0000000111 = 第 7 瓶
```

### 答案

**用 10 只白鼠，每只白鼠代表二进制的一位，通过死掉的白鼠组合出毒药的编号。**

## 三、老板分金条

### 问题

你有一根金条，需要分 7 天给工人付工资。每天付 1/7，但只能切 2 刀。怎么切？

### 思路

关键：工人可以找零。

**切法**：切成 1/7、2/7、4/7 三段。

```
第 1 天：给 1/7                              （工人持有：1/7）
第 2 天：给 2/7，工人找回 1/7                 （工人持有：2/7）
第 3 天：给 1/7                              （工人持有：1/7 + 2/7 = 3/7）
第 4 天：给 4/7，工人找回 1/7 + 2/7          （工人持有：4/7）
第 5 天：给 1/7                              （工人持有：1/7 + 4/7 = 5/7）
第 6 天：给 2/7，工人找回 1/7                （工人持有：2/7 + 4/7 = 6/7）
第 7 天：给 1/7                              （工人持有：1/7 + 2/7 + 4/7 = 7/7）
```

### 答案

**切成 1、2、4 三段（1:2:4 比例），通过找零组合出每天的工资。** 这就是二进制的思想。

## 四、高楼扔鸡蛋

### 问题

有一栋 N 层的楼，有 2 个鸡蛋。鸡蛋在第 X 层及以上扔下来会碎，在第 X 层以下不会碎。最坏情况下，最少需要扔几次才能确定 X？

### 思路

**动态规划**。

设 `dp[n][k]` 表示 n 个鸡蛋、k 次尝试最多能检测多少层楼。

状态转移方程：

```
dp[n][k] = dp[n-1][k-1] + dp[n][k-1] + 1

含义：
- 鸡蛋碎了：在当前层以下检测 dp[n-1][k-1] 层
- 鸡蛋没碎：在当前层以上检测 dp[n][k-1] 层
- +1 是当前层本身
```

**2 个鸡蛋、N 层楼的最优策略**：

从第 X 层开始扔（X 待定）：
- 如果碎了：从第 1 层到第 X-1 层逐层试，最多 X-1 次 + 1 次 = X 次
- 如果没碎：从第 X + (X-1) 层开始扔（因为只剩 X-1 次机会了）

```
第 1 次：第 X 层
第 2 次：第 X + (X-1) 层
第 3 次：第 X + (X-1) + (X-2) 层
...
总层数 = X + (X-1) + (X-2) + ... + 1 = X(X+1)/2 ≥ N
```

对于 N = 100：X(X+1)/2 ≥ 100 → X ≈ 14

### 答案

**2 个鸡蛋、100 层楼，最坏情况最少需要 14 次。** 第一次从第 14 层开始扔，然后根据结果递减步长。

## 五、火枪手决斗

### 问题

三个火枪手 A、B、C 决斗。A 的命中率 30%，B 的命中率 50%，C 的命中率 100%。三人轮流开枪，A 先手。A 应该先打谁？

### 思路

**关键分析**：

- A 先打 B：如果打中 B，接下来 A 和 C 对决，C 先手，A 必死
- A 先打 C：如果打中 C，接下来 A 和 B 对决，B 先手，A 有一定生存概率
- A 对天开枪：不打任何人，让 B 和 C 先对决

**策略分析**：

1. **A 先打 B**：A 打中 B 的概率 30%，之后 A 对 C（C 先手），A 必死。A 存活概率极低。

2. **A 先打 C**：A 打中 C 的概率 30%，之后 A 对 B。
   - A 对 B：B 先手，B 打中 A 的概率 50%
   - 如果 B 没打中，A 打中 B 的概率 30%
   - A 的存活概率需要计算

3. **A 对天开枪（最优策略）**：
   - 接下来 B 和 C 对决
   - B 会先打 C（C 威胁最大）
   - B 打中 C 的概率 50%
   - 如果 B 打中 C：A 对 B，B 先手
   - 如果 B 没打中 C：C 必定打死 B，然后 A 对 C，A 先手

**A 对天开枪后的存活概率**：

```
情况 1（50%）：B 打死 C → A 对 B，B 先手
  B 打中 A 的概率 50%
  B 没打中（50%）→ A 打中 B 的概率 30%
  A 存活概率 ≈ 50% × (1-50%) × 30% + ...（需要迭代计算）

情况 2（50%）：B 没打死 C → C 打死 B → A 对 C，A 先手
  A 打中 C 的概率 30%
  如果 A 没打中（70%），C 必定打死 A
  A 存活概率 = 30%
```

### 答案

**A 应该对天开枪（故意打空）。** 这样 B 和 C 会先互相攻击，A 渔翁得利。对天开枪的存活概率高于直接打 B 或 C。

## 六、水桶取 X 升水

### 问题

有两个水桶，容量分别是 3 升和 5 升，如何精确量出 4 升水？

### 思路

```
步骤 1：装满 5 升桶              （5升桶=5, 3升桶=0）
步骤 2：用 5 升桶倒满 3 升桶     （5升桶=2, 3升桶=3）
步骤 3：倒掉 3 升桶              （5升桶=2, 3升桶=0）
步骤 4：将 5 升桶的 2 升倒入 3 升桶（5升桶=0, 3升桶=2）
步骤 5：装满 5 升桶              （5升桶=5, 3升桶=2）
步骤 6：用 5 升桶倒满 3 升桶     （5升桶=4, 3升桶=3）
→ 5 升桶中剩余 4 升
```

### 答案

**按照上述 6 步操作，最终 5 升桶中就是 4 升水。**

## 七、最快的 X 匹马

### 问题

25 匹马，5 条赛道（每次最多 5 匹马同时比赛，没有计时器）。最少几次比赛能找出最快的 3 匹马？

### 思路

**第一步：分组赛（5 次）**

```
将 25 匹马分成 5 组（A、B、C、D、E），每组 5 匹
每组比赛一次，得到每组的排名

A 组：A1 > A2 > A3 > A4 > A5
B 组：B1 > B2 > B3 > B4 > B5
...
E 组：E1 > E2 > E3 > E4 > E5
```

淘汰每组的第 4、5 名（共 10 匹），剩下 15 匹。

**第二步：各组第一名赛（第 6 次）**

```
A1, B1, C1, D1, E1 比赛
假设结果：A1 > B1 > C1 > D1 > E1
→ A1 就是最快的马

淘汰：D 组和 E 组所有马（D1 都排第 4，D 组其他更慢）
淘汰：C 组的 C3（C1 第 3，C2 最多第 4，C3 不可能前 3）
淘汰：B 组的 B3（B1 第 2，B2 最多第 3，B3 不可能前 3）
```

**第三步：争夺第 2、3 名（第 7 次）**

剩下候选马：A2, A3, B1, B2, C1（共 5 匹）

比赛这 5 匹，取前 2 名（加上 A1 就是最快的 3 匹）。

### 答案

**最少需要 7 次比赛。**

## 八、喊数抢 30 必胜策略

### 问题

两人轮流喊数，每次可以喊 1~3 个连续数字，谁先喊到 30 谁赢。先手有没有必胜策略？

### 思路

**倒推分析**：

- 喊到 30 的人赢 → 喊到 29 的人输（对手喊 1 个就到 30）
- 喊到 26 的人赢（无论对手喊 1/2/3，自己都能喊到 30）
- 关键数字：30, 26, 22, 18, 14, 10, 6, 2

**规律**：每 4 个数字一个周期（1+3=4），关键数字是 4n+2。

**先手策略**：先喊 1 和 2（喊到 2），之后每次喊的数字个数 = 4 - 对手喊的个数。

```
先手喊：1, 2           → 到达 2
对手喊 1 个（到 3）→ 后手喊 3 个（到 6）
对手喊 2 个（到 4）→ 后手喊 2 个（到 6）
对手喊 3 个（到 5）→ 后手喊 1 个（到 6）
...
最终到达 30
```

### 答案

**先手有必胜策略：先喊 1 和 2，然后每次与对手喊的数字凑成 4（如对手喊 1 个自己就喊 3 个）。** 这样就能依次到达 2, 6, 10, 14, 18, 22, 26, 30。

如果规则是"喊到 30 的人输"，则先手应该先喊 1 个数字到 1，然后同样凑 4 的策略，最终让对手喊到 30。

## 九、取最后一本书必胜策略

### 问题

有 N 本书，两人轮流取，每次可以取 1~M 本。谁取最后一本谁赢。先手有必胜策略吗？

### 思路

**关键**：控制每轮两人共取 M+1 本。

- 先手取 X 本，使得剩余数量是 M+1 的倍数
- 之后无论对手取 K 本，自己就取 M+1-K 本
- 最终先手一定取到最后一本

**公式**：先手取 `N % (M+1)` 本。如果结果为 0，则先手没有必胜策略。

### 答案

**先手先取 `N mod (M+1)` 本，之后每轮与对手凑成 M+1 本。** 如果 N 是 M+1 的倍数，先手必输（对手用同样策略）。

## 十、取石子必胜策略（Nim 游戏）

### 问题

有 3 堆石子，分别有 A、B、C 个。两人轮流从任意一堆取任意个（至少 1 个）。谁取到最后一个谁赢。

### 思路

**异或定理**：将所有堆的数量做异或运算。

```
XOR = A ⊕ B ⊕ C
```

- 如果 XOR ≠ 0，先手必胜
- 如果 XOR = 0，先手必输

**先手策略**：从某一堆中取若干个，使得剩余数量的异或值为 0。

**举例**：

```
3 堆石子：3, 5, 7
XOR = 3 ⊕ 5 ⊕ 7 = 011 ⊕ 101 ⊕ 111 = 001 ≠ 0
→ 先手必胜

先手从第 3 堆取 6 个（剩余 1 个）
新状态：3, 5, 1
XOR = 3 ⊕ 5 ⊕ 1 = 011 ⊕ 101 ⊕ 001 = 111 ≠ 0... 

重新计算：3 = 011, 5 = 101, 7 = 111
3 ⊕ 5 = 110
110 ⊕ 7 = 110 ⊕ 111 = 001

先手需要让 XOR = 0
从第 3 堆取 x 个，使得 3 ⊕ 5 ⊕ (7-x) = 0
3 ⊕ 5 = 6
需要 7-x = 6，x = 1
取 1 个后：3, 5, 6
3 ⊕ 5 ⊕ 6 = 011 ⊕ 101 ⊕ 110 = 000 ✓
```

### 答案

**计算所有堆数量的异或值。非零则先手必胜，零则先手必输。先手的策略是每次取完后让异或值为 0。**

## 十一、乒乓球称重

### 问题

有 12 个乒乓球，其中 1 个重量异常（不知道是轻还是重）。有一个天平，最少称几次能找到异常的那个球？

### 思路

**三分法**。

每次称重有 3 种结果：左倾、右倾、平衡。所以每次称重可以将搜索空间缩小到 1/3。

**12 个球的信息量**：12 个球 × 2 种状态（轻或重）= 24 种可能

**3 次称重的信息量**：3^3 = 27 > 24

所以理论最少 3 次即可。

**具体步骤**：

```
第 1 次：将 12 个球分成 3 组（1-4, 5-8, 9-12）
  称 1-4 vs 5-8

情况 1：平衡 → 异常球在 9-12 中
  第 2 次：称 9,10,11 vs 1,2,3（已知正常）
    平衡 → 12 是异常球
      第 3 次：称 12 vs 1，确定轻重
    左倾 → 9,10,11 中有一个偏重
      第 3 次：称 9 vs 10
        平衡 → 11 偏重
        左倾 → 9 偏重
        右倾 → 10 偏重
    右倾 → 9,10,11 中有一个偏轻（同理）

情况 2：左倾（1-4 偏重或 5-8 偏轻）
  第 2 次：重新分组称重...
  （略，需要详细的分组策略）
```

### 答案

**最少称 3 次。** 关键是利用天平的三态特性（左倾、右倾、平衡），每次将搜索空间缩小到 1/3。
